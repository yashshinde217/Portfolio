# My Portfolio

A personal portfolio website built with **React**, **Three.js**, and **Tailwind CSS** — featuring interactive 3D models, smooth animations, and a contact form powered by EmailJS.

## Preview

![Portfolio Preview](./src/assets/images/hero.jpg)

> Live demo → deployed on [Vercel](https://vercel.com)

## Sections

- **Home** — Interactive 3D island scene with a flying plane and animated sky
- **About** — Brief intro about me and my skills
- **Projects** — Showcase of my work with links
- **Contact** — Contact form powered by EmailJS with a 3D fox animation

## Tech Stack

- React 18
- Three.js + React Three Fiber
- Tailwind CSS
- Vite
- EmailJS
